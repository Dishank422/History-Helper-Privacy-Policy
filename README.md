# History-Helper-Privacy-Policy

History Helper is a Chrome Extension built for research purposes. As part of our research, we collect user browsing history. This includes the webpages and their content on the 3 websites: Proquest, JSTOR and HathiTrust.

As part of our research, we allow the use of this extension only for users who are part of our study. The use of the data will be in accordance with the agreement signed with the users proir to their usage of our extension.
